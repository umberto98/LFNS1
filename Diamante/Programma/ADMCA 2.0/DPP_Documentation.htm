DP4/PX4 Additional Software/Hardware Information Sources

Some of the following information sources may also be included your DP4/PX4. Please check the CD provided with your DP4/PX4 for details.

  • The DP4 Programmer's Guide provides addition information for DP4 programmers.

  • The DP4 Hardware Demonstration Software Project (DP4.vbp) provides detailed DP4 programming details in easy-to-understand Visual Basic.

  • The PX4 Programmer's Guide provides addition information for PX4 programmers.

  • The PX4 Hardware Demonstration Software Project (PX4.vbp) provides detailed PX4 programming details in easy-to-understand Visual Basic.

  • The DP4/PX4 Digital Pulse Processor API (DPPAPI) provides details about programming applications for the DP4/PX4. The DPPAPI includes an Application Programming Interface, Visual Basic and Visual C++ source code examples, and DPPAPI reference documentation.
